# Crystalward

Local co-op **action tower defense** — wardens protect a sacred crystal from the soft dark.

Tone: *The Dark Crystal* × *Legend* (1980s dark fantasy). Pacing: closer to action RTS (high APM) than casual stand-and-build.

**Players:** 1–4 local · **any player can do any job**  
**Target:** MacBook + Nintendo Switch Pro / Joy-Con controllers
